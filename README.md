# SAE-2.01---Developpement-d-une-application
SAÉ 2.01 - Développement d'une application

Groupe Composé de :
Fadi SOLTANI
Lucas MAINGUY
Idir NAIT-ALI
Clement TRASSY

Possibilité de changer les combinaisons : le joueur peut choisir lui meme le nombre de carte du même type dans le mode classique (il peut choisir par exemple que chaque carte possède 5 homonymes)

Mode vie limité : le joueur dispose d'un nombre de vie limité et non d'un timer pour trouver les bonnes paires. Ce dernier verra les cartes au début de la partie pendant 20 secondes.

Mode speedrun : Un timer de 1min est lancé et le joueur doit completer le memory dans ce temps imparti. Une fois complété le joueur gagne du temps suplémentaire (10-15s) et recommence sur une nouvelle grille de memory.
